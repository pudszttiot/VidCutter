
       VidCutter for Windows XP, 2003, Vista, Win 7, 8, 10 (32 and 64 bit)

                                           Version 1.0 Released June 09, 2023
========================================================================


Description
-----------
This is a Python GUI application that allows you to cut videos based on specified start and end times. The application utilizes the Tkinter library for the graphical user interface (GUI) components.
  

Installation
------------
 1. Double-click "VidCutter.setup" to start the installation process
 2. Follow the on-screen instructions
 3. Wait for the installation to complete
 4. Launch the software

That's it! You have successfully installed VidCutter on your computer. I hope this guide was helpful, and feel free to ask if you have any questions or need further assistance.


Usage
--------
  1. Run the executable file (.exe) to launch the VidCutter application.
  2. The main window of the application will appear.
  3. Follow these steps to perform video cutting:

   - Select an input video file by clicking the "Select Input File" button. Supported video formats are .mp4, .avi, and .mkv.
   - Choose an output file name and location by clicking the "Select Output File" button. The default extension is .mp4.
   - Specify the start time and end time for the video segment you want to extract. Use the format "hh:mm:ss.mmm"      (hours:minutes:seconds.milliseconds). The default values are both set to "00:00:00.000".
   - Click the "Cut Video" button to extract the specified segment from the input video and save it to the output file.
   - Alternatively, you can click the "Preview Cut" button to preview the specified segment without saving it.


Uninstallation
--------------
Run Uninstall from 'VidCutter' folder in the Start menu or go to 
Control Panel, launch "Add or Remove Program" applet and find the program here.
  
  Please note, you do not need to unistall the program before installation of a 
  new version! Just install it over an old version.
